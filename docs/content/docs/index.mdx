## What is BuddyPie?

BuddyPie launches Daytona sandboxes, clones a repository, and boots an embedded OpenCode workspace for task-specific AI sessions. It combines infrastructure orchestration with configurable AI workflows to automate development tasks.

## Quick Start

### Prerequisites

- Node.js 22 or later
- Bun runtime
- A Convex deployment
- A Daytona API key
- Clerk application for authentication

### Environment Setup

Create a `.env.local` file in the project root:

```bash
# Required
CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
CLERK_JWT_ISSUER_DOMAIN=https://your-clerk-app.clerk.accounts.dev
VITE_CONVEX_URL=https://your-deployment.convex.cloud
DAYTONA_API_KEY=your-daytona-api-key

# Optional
DAYTONA_API_URL=https://app.daytona.io/api
CONVEX_SITE_URL=https://your-deployment.convex.site
OPENROUTER_API_KEY=sk-or-...
VENICE_API_KEY=your-venice-key
```

### Development Server

```bash
# Install dependencies
bun install

# Start Convex sync and dev server
bun run dev
```

This runs Convex schema sync once, then starts both the web server and Convex dev sync in parallel.

<Callout title="First Run">
On first run, you need to run `bunx convex dev --once` separately to push the schema before the dev server can function properly.
</Callout>

## Workflow Presets

BuddyPie manages OpenCode presets for different work types. Each preset configures AI behavior, skills, and workspace setup.

| Preset | Default Model | Use Case |
|--------|---------------|----------|
| General Engineer | OpenRouter / MiniMax M2.7 | Full-stack product work, refactors, features |
| Frontend Builder | OpenRouter / MiniMax M2.7 | UI components, styling, accessibility |
| Docs Writer | Venice / MiniMax M2.7 | Documentation, guides, README files |

## Architecture Overview

BuddyPie is built on three main components:

1. **Frontend** - TanStack Start React app with Clerk authentication
2. **Backend** - Convex for real-time data, billing, and orchestration
3. **Infrastructure** - Daytona for sandbox provisioning, OpenCode for AI
